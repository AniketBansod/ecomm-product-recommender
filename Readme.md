# Ecomm Product Recommender (AI Powered)

A hybrid e-commerce recommendation engine with:
- Vector embeddings (MiniLM)
- Behavior-based scoring
- LLM explanations (GPT-4o Mini / Gemini / HF)
- Node.js backend + FastAPI recommender microservice
- React + Tailwind frontend

## Project Structure
- /frontend — User interface
- /backend — Main API (Node.js + Express + MongoDB)
- /recommender — AI/ML microservice (FastAPI + FAISS)
- /data — Product datasets
- /scripts — Utility scripts
- /docs — Project documentation

## Branch Strategy
- main → stable build
- dev → active development
